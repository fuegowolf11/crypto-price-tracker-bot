# Crypto Price Tracker & Alert Bot

A simple Python bot that fetches live cryptocurrency prices from CoinGecko and displays them in the terminal.

![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)
![CoinGecko](https://img.shields.io/badge/CoinGecko-0082C8?style=for-the-badge&logo=coingecko&logoColor=white)

## ✨ Features
- Real-time price fetching (Bitcoin, Ethereum, Solana, etc.)
- Clean terminal output with timestamps
- Easy to extend with alerts and notifications
- Configurable via `.env`

## 🛠 Tech Stack
- Python 3
- pycoingecko (CoinGecko API)
- python-dotenv
